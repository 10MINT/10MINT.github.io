All About Start Values

(At least I hope it's all)
For help see the reference

There are two types of start values

  • start value
  • plain start text

The plain start text should always be used when communicating with other apps. You can send and receive values from other apps with it! For all other purposes you should use the start value. Use it e.g. when passing values from one screen to another.

When the screen closes, the start value does not change! Use the variable result from Screen.Other Screen Closed if you pass values through Close screen with value!

According to Taifun there are currently some issues of the Companion.


The close screen with value block triggers both the Initialize and OtherScreenClosed events instead of only the OtherScreenClosed event.
